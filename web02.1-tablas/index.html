Tablas.

La etiqueta table define una tabla

th Define el encabezado de la tabla

tr Define una fila de una tabla

td Define la celda de una tabla

caption Define el título de una celda

colgroup Especifica un grupo de una o más columnas para aplicar formato

thead Define la cabecera de una tabla

tbody Define el cuerpo de una tabla

tfoot Define el pie de una tabla

Rowspan, indica el número de filas que ocupará la celda

Colspan, indica el número de columnas que ocupará la celda.
